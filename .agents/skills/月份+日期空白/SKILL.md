# 批次抹除月份與日期數字技能 (月份+日期空白)

本技能可用於自動化批量「抹除」工地或呈報照片中的白板「月份」與「日期（日）」數字，使月份與日期欄位同時完全留白（淨空），同時完美保留白板的底線、網格與原有材質質感。

## 適用場景
當需要將某個資料夾下多張白板照片中的月份（如「1月」）與日期（如「13日」）同時抹除淨空，以便後續手動填寫或保持空白時。

## 技能資源
本技能附帶一個批次消除影像處理 Python 腳本：
* `erase_date.py`：主執行程式。

## 參數與變數說明
呼叫本程式時，可透過命令列參數傳入以下變數：

| 參數 | 必要 | 說明 | 範例 / 預設值 |
|---|---|---|---|
| `--folder` | **是** | 包含待修改 JPG 圖片的資料夾路徑 | `測試SKILLS_08` |
| `--ref-image` | 否 | 包含基準年份、原月份與原日期範本的參考圖片路徑 | 預設自動搜尋資料夾中第一個檔案 |
| `--year-bbox` | 否 | 年份模板在參考圖片中的 ymin,ymax,xmin,xmax 座標 | `595,625,155,210` |
| `--month-bbox` | 否 | 月份模板在參考圖片中的 ymin,ymax,xmin,xmax 座標 | `595,620,235,260` |
| `--day-bbox` | 否 | 日期模板在參考圖片中的 ymin,ymax,xmin,xmax 座標 | `595,620,290,325` |
| `--year-thresh` | 否 | 年份比對的信心度門檻 (0.0 ~ 1.0) | 預設為 `0.70` |
| `--month-thresh` | 否 | 月份比對的信心度門檻 (0.0 ~ 1.0) | 預設為 `0.75` |
| `--day-thresh` | 否 | 日期比對的信心度門檻 (0.0 ~ 1.0) | 預設為 `0.75` |

## 執行步驟說明

### 步驟 1：準備環境與相依套件
執行此腳本前，需確保系統已安裝以下 Python 套件：
```bash
pip install opencv-python numpy
```

### 步驟 2：執行批次抹除命令
開啟終端機並切換至專案根目錄，執行下方指令即可批次淨空月份與日期欄位：

```bash
python .agents/skills/月份+日期空白/erase_date.py --folder "測試SKILLS_08_雙空白" --ref-image "測試SKILLS_08_雙空白\LINE_ALBUM_頂崁里廢棄物呈報_260128_2.jpg" --year-bbox "595,625,155,210" --month-bbox "595,620,235,260" --day-bbox "595,620,290,325"
```

## 注意事項與調優建議
1. **舊字體「抹除範圍」不夠大？**：如果原字跡筆劃較粗，抹除後留有邊緣陰影，可直接使用文字編輯器打開 `erase_date.py`，調整 `iterations=2` 為 `3` 或 `4`。
2. **無痕覆蓋原理**：本程式採用影像局部修補演算法（Inpainting），配合自適應 Otsu 筆跡遮罩，能夠完美避開白板上的橫向或縱向網格線，確保抹除後的區域網格自然延伸且無人工塗抹感。
